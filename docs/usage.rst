Usage Scenarios
===============

The page lists some usage scenarios.

Each scenario can be run in one of two modes. Pick one and use the matching script in every step:

* ``rmw_zenoh``: use ``rmw_zenoh_cpp`` directly as the RMW.
* ``ros2dds``: use ``rmw_cyclonedds_cpp`` with ``zenoh-bridge-ros2dds``.
