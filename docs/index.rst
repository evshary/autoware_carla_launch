autoware_carla_launch
=====================


The package includes launch file to run Autoware, Carla agent, and bridges.

* `zenoh-bridge-dds <https://github.com/eclipse-zenoh/zenoh-plugin-dds>`_
* `zenoh-bridge-ros2dds <https://github.com/eclipse-zenoh/zenoh-plugin-ros2dds>`_
* `zenoh_carla_bridge <https://github.com/evshary/zenoh_carla_bridge>`_
